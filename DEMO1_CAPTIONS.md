## Caption styling tips

- Keep each line **on screen 3–4s**; if a line feels long, split it in two.
- **Bottom-center**, semi-transparent dark background bar, white text — readable over the dark UI.
- Don't caption *every* second — silence between captions lets the visuals (and the lightning on the wires) breathe.
- The 3 captions that matter most: **1:22** (what the agent does), **2:02** (REAL transfer), **2:14** (anchored on-chain). Make sure those land clearly.
